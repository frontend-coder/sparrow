Посадка HTML шаблона Sparrow


Скачан бесплатный шаблона Sparrow, предусмотены такие страницы:

главная в виде landing page
о нас - размещена информация о участинках команды/работниках компании, фирты
наш блог (категории) - страница категорий, меток, поиска, некоторых других
наш блог (одиничные записи) - страница одиночных записей
наше портфолио - представляет собой 2 шаблона под одноименные страницы.

Установка и тестирование темы Sparrow WordPress возможна при достижении до версии v.1.00
